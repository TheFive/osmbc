      Filter Explanation

      Disable Written Articles

      This filter disables all articles, that have already a text in the main (=left) language. Use this filter to come quick to the articles, that needs to be written to be published.

      Disable More Actual Edits

      This filter disables all articles, where the markdown on the left hand is newer than the markdown on the right hand. Use this, to check wether there are relevant changes in the markdown on the right column during translation.

      Disable Own Collections

      Use this to disable all collections, where you have contributed to.

      Disable Other Collections

      This can be used to focus on your own collections, so that you can easily add your Ideas from collection process.
